Game's Rules

Setup

  • Players take turns guessing letters in the word.
  • If a guessed letter is correct, fill it in the blank spaces.
  • If a guessed letter is incorrect, start drawing the hangman.
  • Limited incorrect guesses (5).
  • The game ends when
  • Players guess the word correctly (they win).
  • The hangman is fully drawn (the guesser loses).
  • Reveal the word if players guess it correctly.
